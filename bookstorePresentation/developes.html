Σχετικά με το Έργο μας!!

Οι Τεχνολογίες που χρησιμοποιήσαμε:

  • React για τη δημιουργία της διεπαφής
  • Vite για τη δημιουργία των production και development versions του frontend
  • MySQL για την αποθήκευση των δεδομένων των βιβλίων
  • Node.js και npm για τη διαχείριση των dependences και την εκτέλεση του server
  • Apache Tomcat για το hosting της εφαρμογής

Workflow

Περιβάλλον

  • Εγκατάσταση του Node.js και npm για τη διαχείριση των dependences
  • Διαμόρφωση του Vite με το plugin για React

Development Workflow

Ανάπτυξη Frontend:

  • Δημιουργήσαμε components και φτίαξαμε το user interface με React
  • Δημιουργήσαμε entry point στο index.html που αναφέρεται στο αρχείο src/main.jsx, το οποίο αρχικοποιεί την εφαρμογή React
  • Κάναμε το styling χρησιμοποιώντας αρχεία CSS στον κατάλογο src/styles

Βάση Δεδομένων Backend:

  • Σχεδιάσαμε μια βάση δεδομένων MySQL με έναν πίνακα books για την αποθήκευση δεδομένων
  • Διαμορφώσαμε τον backend server για σύνδεση με τη βάση δεδομένων

Production Workflow

  • Δημιουργήσαμε την εφαρμογή για production χρησιμοποιώντας την εντολή npm run build, που φτιάχνει τον φάκελο dist
  • Αντιγράψαμε τον φάκελο dist στον Tomcat
  • Διαμορφώσαμε τον Tomcat για να εξυπηρετεί τα στατικά αρχεία, επιτρέποντας την πρόσβαση στην εφαρμογή μέσω ενός browser
